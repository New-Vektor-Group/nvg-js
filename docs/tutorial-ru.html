Main

nvg-main.js
  • Высота экрана

    var y = nvg_sh(); :int

    Здесь y может быть, например, 1080 пикселей. (Работает при загрузке страницы)
  • Ширина экрана

    var x = nvg_sw(); :int

    Здесь x может быть, например, 1920 пикселей. (Работает при загрузке страницы)
  • Высота окна (сайта)

    var y = nvg_sha(); :int

    Здесь y может быть, например, 929 пикселей. (Работает при загрузке страницы)
  • Ширина окна (сайта)

    var x = nvg_swa(); :int

    Здесь x может быть, например, 927 пикселей. (Работает при загрузке страницы)
  • Узнать позицию скролла

    var y = nvg_scroll(); :int

    Здесь x может быть, например, 0. (Работает при загрузке страницы)
  • Функция спускания вниз/вверх !только для overflow scroll элементов!

    nvgjs.scrollbot(element);

    nvgjs.scrolltop(element);

    Например, (HTML)
    <textarea id="lbtn">МНОГО.ОЧЕНЬ.МНОГО.ТЕКСТА</textarea>
    .(JS).
    nvgjs.scrollbot("#lbtn");
  • Функция плавного прокручивания вверх/вниз/по выбору

    nvgjs.ScrollTopSoft();

    Например, (HTML)
    <button onclick="nvgjs.ScrollTopSoft()" class="btn">Вверх</button>

    nvgjs.ScrollBotSoft();

    Например, (HTML)
    <button onclick="nvgjs.ScrollBotSoft()" class="btn">Вниз</button>

    nvgjs.ScrollSoft(y-coord);

    Например, (HTML)
    <button onclick="nvgjs.ScrollSoft(200)" class="btn">На 200</button>
  • Автоматически скопировать текст кликом

    nvgcopy(button, text_area);

    Установить кнопку, на которую если нажать, то скопируется тот или иной текст.
    Например, (HTML)
    <button id="scrbtn" class="btn">Scroll</button>
    .(JS).
    nvgcopy("btn", "text");

Math

nvg-math.js
  • Рандом

    var cis = nvgRand(min, max); :int

    Возвращает случайное числом между минимумом (включительно) и максимумом (не включительно)
  • Сигмоида

    var cis = nvgm.sigm(x); :float

    Возвращает результат логистической функции.
























Работа с cookies

nvg-cookie.js
  • Создать

    nvg_cook.setCookie(keyname, value, options);

    Например, nvg_cook.setCookie('lol',228, {expires: 50});
    expires это время жизни куки (секунды).
  • Запросить

    nvg_cook.getCookie(keyname); :string

  • Удалить

    nvg_cook.deleteCookie(keyname);

















Работа с текстом

nvg-text.js
  • Переносы строчек

    Возвращает строку с тегами <br/> или же <br>, где по сути стоят (\r\n, \n\r, \n and \r).
    Например, $("#text").html(nvgtext.nl2br($("#text").text(),false));. Обработка текста по функции nl2br.
















Работа с падающими частицами

nvg-snow.js

Для смены конфигурации нужно сначала изменить её, а потом обновить (или перерисовать).

  • Инициация

    var sfall = new nvg_snow();

    Это конструктор. Здесь настройки по дефолту для снежинок. Запускается сразу.
  • Стоп

    sfall.stop();

    Остановить движение.
  • Стереть

    sfall.hide();

    Скрыть(стереть) частицы.
  • Нарисовать

    sfall.draw();

    Установить позиции и нарисовать частицы на экране.
  • Старт анимации

    sfall.init();

    Начать и проигрывать анимацию (падение).
  • Перерисовать

    sfall.redraw();

    Обновить настройки анимации и нарисовать частицы.
  • Обновить

    sfall.update();

    Перерисовать, обновить и начать анимацию
  • Удалить

    sfall.delete();

    Скрыть, остановить анимацию и удалить частицы.
  • Конфигурация настроек
    Здесь то, что стоит по умолчанию:

    sfall.speed = 50; //Скорость прорисовки.

    sfall.snowmax = 30; //Максимальное число частиц на экране.

    sfall.snowletter = "*"; //Что рендерить? * хорош для снежинок.Но вы можете вписать и изображение тегами img, svg или другие элементы.

    sfall.snowcolor = new Array("#b9dff5","#7fc7ff","#7fb1ff","#7fc7ff","#b9dff5"); //Все возможные цвета в массиве для частиц.

    sfall.snowmaxsize = 50; //Максимальный размер частиц.

    sfall.snowminsize = 25; //Минимальный размер частиц.

    sfall.snowingzone = 1; //Где будет анимация? 1. Весь экран, 2. Левая часть, 3. Центр, 4. Правая часть

    sfall.snowtype = new Array("Times"); //Шрифт

















Работа со стилями

nvg-css.js
  • Плавный показ элемента относительно позиции скролла

    var element_soft= new nvgs(element, offset, direction, offset2 (не важно));

    Установить элемент, который будет показывать/скрываться в зависимости от текущей позиции скролла.
    Например, (HTML)
    <button id="scrbtn" class="btn">Scroll</button>
    .(JS).
    var scrbtn = new nvgs("#scrbtn",90,1); // Эта кнопка появится на экране когда скролл будет ниже установленной координаты
    var scrbtn = new nvgs("#scrbtn",90,0); // Эта кнопка появится на экране когда скролл будет выше установленной координаты
    var scrbtn = new nvgs("#scrbtn",150,2,250); // Эта кнопка появится на экране когда скролл будет выше 150, но ниже 250.
  • Зафиксировать положение элемента

    nvgs.setFix(element, position,x offset,y offset);

    ПОЗИЦИИ: tl- Вверху слева, tr- Вверху справа, bl- Внизу слева, tr- Внизу справа
    Например, (HTML)
    <button id="scrbtn" class="btn">Scroll</button>
    .(JS).
    nvgs.setFix("#scrbtn", "br", 20, 20);
















Работа с изображениями

nvg-img.js
  • Вставить картинки с буфера обмена

    nvgi.paste = true;   //Инициация, то есть начать отслеживание момента вставок.

    nvgi.needToS = true; nvgi.ToS = "element";   //Чтобы показывать (отображать) изображение на странице

    Вы можете скопировать любое изображение и вставить его где угодно (или загрузить, запостить) нажатием мышки или клавиш CTRL + V
    Например, (HTML)
    <div id="pst1"> </div>
    .(JS).
    nvgi.paste = true;
    nvgi.needToS = true;
    nvgi.needToS = "#pst1";
    .(Результат).
    <div id="pst1"> <img src="blob:null5850238304" > </div>
  • Авто создание полного просмотра изображения

    var img = new nvg_modal("img","element","url to image", max width (не важно)); //Ширина не обязательна. Выражена в пикселях.

    Когда Вы нажимаете по элементу, открывается модальное окно с Вашим изображением.
    Вам надо добавить аргумент "img" для работы с изображениями. "img2" также для изображений, но они будут 100% в полном размере на всё окно.
    Например, (HTML)
    <img id="btn1" src="lol.png">
    .(JS).
    var img_img = new nvg_modal("img","#btn1","https://pp.userapi.com/hdUzk.jpg");
