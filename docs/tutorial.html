NVG Guide JavaScript





Main

  • Get height of screen

    var y = nvg_sh;

    Here y might be, for example, 1080 pixels.
  • Get width of screen

    var x = nvg_sw;

    Here x might be, for example, 1920 pixels.
  • Button for scroll action

    scrollTopSoft(id);

    For example, (HTML)
    <button id="scrbtn" class="btn">Scroll<button/>
    .(JS).
    scrollTopSoft(scrtbtn);
  • Sroll button in bottom right

    setBtnSofter(offset, btn_id);

    Set button, which should be to show or to hide in relative to current scroll position.
    For example, (HTML)
    <button id="scrbtn" class="btn">Scroll<button/>
    .(JS).
    setBtnSofter(90, scrbtn);








Math

  • Random

    var cis = nvgRand(min, max);

    Returns a random number between min (included) and max (not included)
