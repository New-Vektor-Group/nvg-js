Getting Started

  1. Download JQuery

  2. Download Bootstrap

  3. Download NVG *lib*

  4. Connect

    5. <script src="YOUR_PATH/nvg-main.js"></script>

    Connect any library

    <script src="YOUR_PATH/nvg-text.js"></script>

    !Connect all libraries at once

    <script src="YOUR_PATH/nvg.js"></script>

  5. Separate script

    6. Next, you should create a separate script tag. There you can use our framework. For example,

    <script type="text/javascript">

    #For example, nvg-text.js
    nl2br(mystring);

    </script>

Main

nvg-main.js
  • Get height of screen

    var y = nvg_sh(); :int

    Here y might be, for example, 1080 pixels. (Works when page loads)
  • Get width of screen

    var x = nvg_sw(); :int

    Here x might be, for example, 1920 pixels. (Works when page loads)
  • Get height of window

    var y = nvg_sha(); :int

    Here y might be, for example, 929 pixels. (Works when page loads)
  • Get width of window

    var x = nvg_swa(); :int

    Here x might be, for example, 927 pixels. (Works when page loads)
  • Get scroll position at a page

    var y = nvg_scroll(); :int

    Here x might be, for example, 0. (Works when page loads)
  • Button for scrolling down/up !only overflow scroll elements!

    nvgjs.scrollbot(element);

    nvgjs.scrolltop(element);

    For example, (HTML)
    <textarea id="lbtn">TEXT.A.LOT.OF.TEXT</textarea>
    .(JS).
    nvgjs.scrollbot("#lbtn");
  • Click for soft scroll to top/bot/custom

    nvgjs.ScrollTopSoft();

    For example, (HTML)
    <button onclick="nvgjs.ScrollTopSoft()" class="btn">Up</button>

    nvgjs.ScrollBotSoft();

    For example, (HTML)
    <button onclick="nvgjs.ScrollBotSoft()" class="btn">Down</button>

    nvgjs.ScrollSoft(y-coord);

    For example, (HTML)
    <button onclick="nvgjs.ScrollSoft(200)" class="btn">Scroll</button>
  • Go to HTTPS

    nvgjs.go_ssl();

  • Auto copy text by clicking

    nvgcopy(button, text_area);

    Set button when click to copy some text.
    For example, (HTML)
    <button id="scrbtn" class="btn">Scroll</button>
    .(JS).
    nvgcopy("btn", "text");

    EXAMPLE


    Some text by NVG. Check it. Ctrl+V somewhere.
  • The visibility of an element on the screen

    nvgjs.isVisible(element) :bool;

Work with cookies

nvg-cookie.js
  • Create

    nvg_cook.setCookie(keyname, value, options);

    For example, nvg_cook.setCookie('lol',228, {expires: 50});
    expires is timelife of cookie (seconds).
  • Get

    nvg_cook.getCookie(keyname); :string

  • Delete

    nvg_cook.deleteCookie(keyname);

















Work with text

nvg-text.js
  • New lines

    nvgtext.nl2br(element, is xhtml?); :string

    Returns string with <br/> or <br> inserted before all newlines (\r\n, \n\r, \n and \r).
    For example, $("#text").html(nvgtext.nl2br($("#text").text(),false));. Process text by function nl2br.
  • No copy!

    nvgtext.nocopy(element);

    Remove the ability to select and copy.
    For example,
    nvgtext.nocopy("#nocopyme");
















Work with falling particles

nvg-snow.js

For change configuration you should change it and after it update (or redraw).

  • Start

    var sfall = new nvg_snow();

    This is constructor. Here are default settings for snow. It runs at once.
  • Stop animation

    sfall.stop();

    Stop movement.
  • Hide

    sfall.hide();

    Hide particles.
  • Draw

    sfall.draw();

    Set init positions and draw particles
  • Start movement

    sfall.init();

    Start and play animation (falling).
  • Redraw

    sfall.redraw();

    Update settings for animation and redraw it.
  • Update

    sfall.update();

    Redraw, update and start animation
  • Delete

    sfall.delete();

    Hide, stop animation and delete particles.
  • Configuration of setting
    Here are defaults:

    sfall.speed = 50; //Set render speed of animation.

    sfall.snowmax = 30; //Maximum of particles on a screen.

    sfall.snowletter = "*"; //What to render? * is a good for snowflake.But you can write and img, svg tags and other elements.

    sfall.snowcolor = new Array("#b9dff5","#7fc7ff","#7fb1ff","#7fc7ff","#b9dff5"); //All colors for particles.

    sfall.snowmaxsize = 50; //Maximum size of a particle.

    sfall.snowminsize = 25; //Minimum size of a particle.

    sfall.snowingzone = 1; //Where to render? 1. All screen, 2. Left part, 3. Center, 4. Right part

    sfall.snowtype = new Array("Times"); //Font

















Work with styles

nvg-css.js
  • Soft showing of element in relative to scroll position

    var element_soft= new nvgs(element, offset, direction, offset2 (not important));

    Set any element, which should to show or to hide in relative to current scroll position.
    For example, (HTML)
    <button id="scrbtn" class="btn">Scroll</button>
    .(JS).
    var scrbtn = new nvgs("#scrbtn",90,1); // This button is showen when scroll page is below offset variable
    var scrbtn = new nvgs("#scrbtn",90,0); // This button is showen when scroll page is above offset variable
    var scrbtn = new nvgs("#scrbtn",150,2,250); // This button is showen when scroll page is above 150, but below 250.
  • Set fixed position for element

    nvgs.setFix(element, position,x offset,y offset);

    POSITIONS: tl- top left, tr- top right, bl- bottom left, tr- bottom right
    For example, (HTML)
    <button id="scrbtn" class="btn">Scroll</button>
    .(JS).
    nvgs.setFix("#scrbtn", "br", 20, 20);
















Work with images

nvg-img.js
  • Paste images from clipboard

    nvgi.paste = true;   //Init, start to listen 'paste' event.

    nvgi.needToS = true; nvgi.ToS = "element";   //For showing an image.

    You can copy any image and paste everywhere (or upload, post) by clicking or pressing CTRL + V
    For example, (HTML)
    <div id="pst1"> </div>
    .(JS).
    nvgi.paste = true;
    nvgi.needToS = true;
    nvgi.needToS = "#pst1";
    .(Result).
    <div id="pst1"> <img src="blob:null5850238304" > </div>
  • Auto create full size modal for image

    var img = new nvg_modal("img","element","url to image", max width (not important)); //width is not required. It has pixels.

    When you click to an element, shows a modal with your image.
    You should add first arg. "img" to work with images. "img2" is for images too, but they are in full size.
    For example, (HTML)
    <img id="btn1" src="lol.png">
    .(JS).
    var img_img = new nvg_modal("img","#btn1","https://pp.userapi.com/hdUzk.jpg");
  • Background image loading

    nvgi.preLoad("elements", "phant_attr" (not important));

    PreLoad images in background.
    For example, add attribute "data-src". Into this attribute add url of real image. In the usual src attribute, add a link to a "fast" image. Then, write
    nvgi.preLoad(".my_images","data-src");
